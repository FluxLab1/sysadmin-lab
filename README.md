# Sysadmin Lab

This repository is a hands-on system administration lab focused on building practical Linux skills through real-world tasks, documentation, and automation.

The goal of this lab is to develop a strong foundation in Linux system administration while applying security best practices and troubleshooting techniques commonly used in production environments.

---

## Objectives

- Develop practical Linux system administration skills
- Practice automation and scripting with shell tools
- Learn system hardening and security fundamentals
- Document configurations, processes, and lessons learned
- Build a reproducible home-lab style environment

---

## Lab Focus Areas

- Linux system configuration and maintenance  
- User, group, and permission management  
- Service and process management  
- System monitoring and logging  
- Automation with shell scripts and cron jobs  
- Basic container usage and tooling  
- Security hardening and audit fundamentals
